Sistema desenvolvido especialmente para a Budega do Zé.

## Objetivo

Criar um sistema completo de gerenciamento para bares e restaurantes, totalmente personalizado para a Budega do Zé.

- Login
- Dashboard
- Comandas
- Cadastro de Clientes
- Cadastro de Produtos
- Controle de Estoque
- Caixa
- Funcionários
- Impressão de Conta
- Relatórios
- Backup
